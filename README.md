# 📡 Telco Churn Analysis

A data analysis project on Telco customer data to identify churn patterns, key risk factors, and retention strategies using SQL, Python, and interactive visualizations.

---

## 🔗 Live Dashboard

👉 [View Live Dashboard](https://gauravkashyap-lab.github.io/Telcho-churn-analysis/)

---

## 🛠️ Tools Used

- **SQL** — Data extraction and querying
- **Python** — Data cleaning and analysis (Pandas, Matplotlib, Seaborn)
- **Power BI** — Business intelligence dashboard
- **HTML / CSS / JavaScript** — Interactive web dashboard
- **Java** — Backend processing

---

## 📊 Key Findings

- Identified key factors contributing to customer churn
- Analyzed relationship between contract type, tenure, and churn rate
- Uncovered patterns in payment methods and service usage among churned customers

---

## 📌 Project Objective

To analyze Telco customer data and identify the key drivers of customer churn, enabling businesses to take proactive retention measures and reduce customer loss.
